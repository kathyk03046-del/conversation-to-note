# Conversation to Note

Turns a specified segment of a conversation into a structured,
self-contained Obsidian note, linked to relevant existing notes
in the vault.

## How to run
Read config.json, then follow skills/orchestrator.md.

## Hard rules
1. Never write to vault without explicit user confirmation
2. Never determine conversation scope without explicit user
   instruction — the user must specify where the conversation
   to archive begins
